# 🎮 Gerenciador de Ligas e Times

> Um sistema de gerenciamento via terminal para organizar times de diferentes categorias de jogos.

## 📋 Sobre o Projeto
Este projeto foi desenvolvido para colocar em prática conceitos fundamentais de programação e lógica estruturada. Trata-se de uma aplicação de terminal construída para cadastrar, listar e gerenciar o status de atividade de times de jogos variados (como RPG, RP, FPS, etc.). 

O sistema mantém os dados na memória durante a execução, permitindo uma navegação contínua através de menus interativos.

## ⚙️ Funcionalidades
- **Cadastrar novos times:** Inserção de nome e categoria do time.
- **Listagem organizada:** Visualização formatada de todos os times cadastrados, mostrando nome, categoria e status atual.
- **Ativar/Desativar:** Alteração do status de atividade (Ativo/Desativado) de um time específico através de um sistema de busca.
- **Interface Limpa:** Uso de recursos do sistema para limpar a tela do terminal entre as operações, garantindo uma melhor experiência visual.

## 💻 Tecnologias Utilizadas
- **Python**
- Biblioteca padrão `os` (para limpeza de tela do terminal)
